วิธีสร้าง Google Sheet Backend (ทำครั้งเดียว):

1. เปิด sheets.new เพื่อสร้าง Google Sheet ใหม่
2. ตั้งชื่อ Sheet เป็น Portfolio Dashboard
3. ไปที่ Extensions → Apps Script
4. ลบโค้ดเดิมออก และวางโค้ดด้านล่างนี้ทั้งหมด:

        
      

      

        5. กด Save แล้วกด Deploy → New Deployment

        6. เลือก Type: Web app, Execute as: Me, Who has access: Anyone

        7. กด Deploy และ Authorize แล้วคัดลอก Web App URL
